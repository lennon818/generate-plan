# Generate Plan

A minimalist polaroid-style web app for generating production plans.

## Features

- Polaroid-style UI design
- Notes input with contact extraction
- Moodboard image upload
- Generate structured production plan
- API integration with localhost:3000

## Usage

1. Open `index.html` in a browser
2. Enter notes in the NOTES field
3. Optionally upload moodboard images
4. Click GENERATE PLAN

## Tech Stack

- HTML, CSS, JavaScript
- Python backend (localhost:3000)
